Операції:
1) +, -, *, /
2) mod, div
 для обчислення остачі від ділення слід використовувати оператор "%"
  наприклад, значенням виразу "5|2" буде число 1
 для обчислення цілої частини від ділення слід використовувати оператор "|"
  наприклад, значенням виразу "5|2" буде число 2
3) ^
4) max(x, y), min(x, y)
 для обчислення максимального з двох чисел слід використовувати оператор ">"
  наприклад, значенням виразу "5>2" буде число 5
 для обчислення мінімального з двох чисел слід використовувати оператор "<"
  наприклад, значенням виразу "5<2" буде число 2
